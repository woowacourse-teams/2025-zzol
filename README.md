## 똥손도 즐기는 운빨 게임!

친구들과 반복되는 '오늘은 누구로 정할까?', 지루하지 않으신가요?

쫄(ZZOL)은 **똥손도 재미있게 즐길 수 있는 실시간 대결 게임 서비스**입니다.

단순한 뽑기, 미니 게임 그리고 룰렛 시스템으로 더욱 재밌는 경험을 제공합니다.

👉🏻[게임하러가기](https://zzol.site)
