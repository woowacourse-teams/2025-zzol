# ADR 인덱스

주요 기술 의사결정 목록. 새 ADR 작성은 `/adr [주제]`로 한다.

코드 작성 시 **영향 범위**와 작업 중인 패키지/주제를 비교한다. 겹치는 항목이 있으면 해당 ADR 파일을 읽어 핵심 제약과 충돌 여부를 확인한다.
